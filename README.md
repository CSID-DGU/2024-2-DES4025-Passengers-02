## 🔎 프로젝트 소개
- 기기의 **NFC 기능**을 통해 **모바일 기기**로 메뉴 주문을 처리할 수 있는 **웹 기반 키오스크 주문 시스템**
- 자신에게 친숙한 모바일 기기에서 VoiceOver, TalkBack과 같은 **접근성 기능**을 활용해
- 시각 장애인에게 **구체적인 음성 안내 및 피드백**을 제공하는 서비스
